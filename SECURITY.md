## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, **do not open a public issue**.

Instead, please report it privately by emailing or contacting me directly through GitHub.

**What to include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes if you have them

## Response Time

I aim to respond to all security reports within **72 hours** and will keep you updated on the fix timeline.

## Disclosure Policy

Once a vulnerability is patched and a fix is released, I'm happy to publicly credit the reporter if they wish. Please allow reasonable time for a fix before any public disclosure.

## Scope

This project is a local penetration testing tool intended for use in **isolated lab environments only**. Any vulnerabilities related to unintended remote access, data exposure, or privilege escalation are taken seriously.
