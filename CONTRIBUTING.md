# Contributing to PenMaster Security

Thanks for your interest in contributing! This project is an autonomous penetration testing agent powered by a local LLM. Contributions from the security, AI, and Python communities are welcome.

## What We're Looking For

- Bug fixes and stability improvements
- New tool integrations (recon, exploitation, reporting)
- Improvements to the agent loop logic
- Documentation improvements
- Testing against additional targets

## How to Contribute

1. Fork the repository
2. Create a new branch for your change
3. Make your changes and test them
4. Open a pull request with a clear description of what you changed and why

## Code Style

- Python 3.x
- Keep it clean and commented
- No hardcoded credentials or target IPs

## Reporting Bugs

Use the GitHub Issues tab. Be specific — include what happened, what you expected, and any error output.

## Questions

Open a GitHub Issue or reach out via the contact method in our Code of Conduct.
